طريقة الاستخدام:

1. افتح ملف index.html للتجربة.
2. اكتب الاسم واضغط تحميل البطاقة.
3. لاستبدال البطاقة التجريبية:
   - احذف card.png الحالي.
   - ضع البطاقة الأصلية بنفس الاسم: card.png
4. إذا كان مكان الاسم غير مناسب، افتح index.html وعدّل هذا السطر:

ctx.fillText(name, canvas.width / 2, 580);

الرقم 580 يتحكم بمكان الاسم عمودياً:
- رقم أقل = الاسم أعلى
- رقم أكبر = الاسم أسفل

يمكن رفع الملفات على GitHub Pages أو Netlify أو أي استضافة داخلية.
